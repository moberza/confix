acltx_prog_jpeg2ps

Synopsis

ACLTX_PROG_JPEG2PS([ACTION-IF-NOT-FOUND])

Description

This macro find a jpeg2ps application and set the variable jpeg2ps to the name of the application or to no if not found if ACTION-IF-NOT-FOUND is not specified, configure fail when then application is not found.

Author

Boretti Mathieu <boretti@eig.unige.ch>

Last Modified

2006-07-16

M4 Source Code 

AC_DEFUN([ACLTX_PROG_JPEG2PS],[
AC_CHECK_PROGS(jpeg2ps,[jpeg2ps],no)
if test $jpeg2ps = "no" ;
then
        ifelse($#,0,[AC_MSG_ERROR([Unable to find the jpeg2ps application])],
        $1)
fi
])
