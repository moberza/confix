ac_c_long_long_

Synopsis

AC_C_LONG_LONG

Description

Provides a test for the existance of the long long int type and defines HAVE_LONG_LONG if it is found.

change: add a comment for the ac_define <guidod>

Author

Caolan McNamara <caolan@skynet.ie>

Last Modified

2001-05-03

M4 Source Code 

AC_DEFUN([AC_C_LONG_LONG_],
[AC_CACHE_CHECK(for long long int, ac_cv_c_long_long,
[if test "$GCC" = yes; then
  ac_cv_c_long_long=yes
  else
        AC_TRY_COMPILE(,[long long int i;],
   ac_cv_c_long_long=yes,
   ac_cv_c_long_long=no)
   fi])
   if test $ac_cv_c_long_long = yes; then
     AC_DEFINE(HAVE_LONG_LONG, 1, [compiler understands long long])
   fi
])
