ac_cxx_exceptions

Synopsis

AC_CXX_EXCEPTIONS

Description

If the C++ compiler supports exceptions handling (try, throw and catch), define HAVE_EXCEPTIONS.

Author

Todd Veldhuizen, Luc Maisonobe <luc@spaceroots.org>

Last Modified

2004-02-04

M4 Source Code 

AC_DEFUN([AC_CXX_EXCEPTIONS],
[AC_CACHE_CHECK(whether the compiler supports exceptions,
ac_cv_cxx_exceptions,
[AC_LANG_SAVE
 AC_LANG_CPLUSPLUS
 AC_TRY_COMPILE(,[try { throw  1; } catch (int i) { return i; }],
 ac_cv_cxx_exceptions=yes, ac_cv_cxx_exceptions=no)
 AC_LANG_RESTORE
])
if test "$ac_cv_cxx_exceptions" = yes; then
  AC_DEFINE(HAVE_EXCEPTIONS,,[define if the compiler supports exceptions])
fi
])
