ac_cxx_have_std

Synopsis

AC_CXX_HAVE_STD

Description

If the compiler supports ISO C++ standard library (i.e., can include the files iostream, map, iomanip and cmath}), define HAVE_STD.

Author

Todd Veldhuizen, Luc Maisonobe <luc@spaceroots.org>

Last Modified

2004-02-04

M4 Source Code 

AC_DEFUN([AC_CXX_HAVE_STD],
[AC_CACHE_CHECK(whether the compiler supports ISO C++ standard library,
ac_cv_cxx_have_std,
[AC_REQUIRE([AC_CXX_NAMESPACES])
 AC_LANG_SAVE
 AC_LANG_CPLUSPLUS
 AC_TRY_COMPILE([#include <iostream>
#include <map>
#include <iomanip>
#include <cmath>
#ifdef HAVE_NAMESPACES
using namespace std;
#endif],[return 0;],
 ac_cv_cxx_have_std=yes, ac_cv_cxx_have_std=no)
 AC_LANG_RESTORE
])
if test "$ac_cv_cxx_have_std" = yes; then
  AC_DEFINE(HAVE_STD,,[define if the compiler supports ISO C++ standard library])
fi
])
