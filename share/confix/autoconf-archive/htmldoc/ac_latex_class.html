ac_latex_class

Obsolete Macro

Replaced by the newer ACLTX_XXX set of macros.

Synopsis

AC_LATEX_CLASSE(<class1>,<var>)

Description

Test if class1 exists and set $var to the right value 

 AC_LATEX_CLASSES([book],book)
 should set $book="yes"

 AC_LATEX_CLASSES(allo,book)
 should set $book="no"

_AC_LATEX_TEST([
\begin{document}

],[ac_cv_latex_class_]translit($1,[-],[_]),$1)
])

])
