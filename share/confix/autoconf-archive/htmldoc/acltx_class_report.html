acltx_class_report

Synopsis

ACLTX_CLASS_REPORT([ACTION-IF-NOT-FOUND])

Description

This macro test if class report is installed and fail (default) with a error message if not

Author

Boretti Mathieu <boretti@eig.unige.ch>

Last Modified

2006-07-16

M4 Source Code 

AC_DEFUN([ACLTX_CLASS_REPORT],[
ACLTX_CLASS(report,report,[],[m4_ifval([$1],[$1],AC_MSG_ERROR([Unable to find the report class]))])
])
